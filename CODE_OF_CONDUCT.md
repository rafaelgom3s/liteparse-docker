# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project you agree to abide by its terms.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a confidential email to the maintainer listed in the [SECURITY.md](SECURITY.md) file.

Reports will be reviewed promptly and handled with discretion.
